About Apple Safari

Safari is a web browser developed by Apple based on the WebKit engine. First released in 2003 with Mac OS X Panther, a mobile version has been included in iOS devices since the introduction of the iPhone in 2007. It is the default browser on Apple devices. A Windows version, now discontinued,[3] was available from 2007 to 2012.

Safari Technology Preview was first released alongside OS X El Capitan 10.11.4. Safari Technology Preview releases include the latest version of WebKit, incorporating Web technologies to be incorporated in future stable releases of Safari, so that developers and users can install the Technology Preview release on a Mac, test those features, and provide feedback.

Safari is a web browser which is well known amongst Mac Apple users.

Advantages

Very simple to start and use and friendly with spell check.

Home page of "Top Sites" which shows a picture/link to your top 12 visited sites.

Homepage of "Bookmark". Looks just like iTunes but shows recently visited sites.

Simple to organize and customize tabs and bars to your specifications.

Fast web browser

Disadvantages

Founded by Apple Inc. therefore limited to Apple products.

Updates to Safari are rare.

Some websites dont support Safari.

Although pop ups are rare, hard to turn off pop ups.

Boring look.
